REQUIREMENTS
 * Max/MSP 5
 * Java 5 or higher (most computers today have this)

-----------------------------------------------------------------------------

INSTALLATION

 (1) Put lib/jruby4max.jar and lib/jruby.jar in:
     Max5/Cycling '74/java/lib
    
 (2) Put the jruby_for_max folder somewhere on your Max file search path.
     You can either:
	 (a) Copy the folder to Max5/patches
	 (b) or add the folder to the search path via Max's Options menu -> File Preferences
       
 (3) Restart Max/MSP

 (4) To get started, open jruby_for_max/jruby.maxhelp
     or add the object [mxj jruby] to a patcher and right click it -> Help
